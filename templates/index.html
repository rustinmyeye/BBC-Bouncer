Token Bouncer

Welcome to the Token Bouncer! This application automatically monitors a wallet for specific token IDs and bounces (sends to another address) any blacklisted tokens found. Checks for the blacklisted token IDs happen every 5 minutes.

How to use:

  1. Enter your mnemonic in the provided field below and click "Set Mnemonic".
  2. Enter the Token ID's you want to bounce (separated by commas) in the field below and click "Set Token IDs".
  3. If you'd like to specify the address the tokens get sent to, change it in the "Enter New Bounce Address" field.
